Jak wrzucić do GitHub:
1. Rozpakuj ZIP.
2. W repo kliknij Add file -> Upload files.
3. Wgraj wszystkie pliki.
4. Commit changes.

Jak wrzucić do Vercel:
1. Połącz repo z GitHub.
2. Import repo na Vercel.
3. Deploy.
4. Otwórz link na iPhonie i dodaj do ekranu.
