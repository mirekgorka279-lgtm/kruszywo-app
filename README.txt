WERSJA PWA – KALKULATOR KRUSZYWA HANDLOWIEC PRO

Zawartość:
- index.html
- manifest.json
- sw.js
- icon-192.png
- icon-512.png

Jak wrzucić na Vercel:
1. Rozpakuj ZIP.
2. Wgraj cały folder albo jego zawartość jako projekt.
3. Po publikacji otwórz link w Safari.
4. Kliknij Udostępnij -> Dodaj do ekranu początkowego.

Po dodaniu do ekranu:
- aplikacja działa jak PWA
- ma ikonę
- może działać offline po pierwszym otwarciu
- zapisuje dane lokalnie w przeglądarce
